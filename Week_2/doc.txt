1.Получить ключ и значение из консоли
storage.py --key key_name --val value

2.Написать функцию которая будет получать(словарь, 
ключь, значение ) и записывать в данный словарь. 
Если по данному ключу есть значение то добавлять 
его в список

3.Функция которая будет открывать файл:
   если файл не существует:
    создает файл и вызывать
   функцию 2 и передавать в нее пустой словарь и записывать
   в json..
   если существует:
	открывать файл считывать с него словарь вызывать
	функцию 2 и передавать в нее считаный словарь и записывать
   в json.
4.функция которая получает ключ и выводит его значение
если ключь не найден выводит None dict.get(key[, default]) - возвращает значение ключа, 
но если его нет, не бросает исключение, а возвращает default
(по умолчанию None).
storage.py --key key_name
value_1, value_2
